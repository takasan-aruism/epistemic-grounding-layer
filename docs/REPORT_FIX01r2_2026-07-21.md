## 5. 次便に向けた実測所見（2件・いずれも先回りの報告）

### (a) 27B 経路は現状そのままでは動かない

r2b の事前授権は行使しなかったが、将来の判断材料として実測した。

```
:8006                 応答なし（非稼働）
workcell-runner@ unit PrivateNetwork=yes / BindReadOnlyPaths=/run/vllm/qwen.sock のみ
run-with-proxy.sh     LOOPBACK_PORT=8005 を固定
```

つまり **packet の `model` / `endpoint` を書き換えるだけでは 27B に到達できない**。
第2 unix socket + proxy unit + unit 改版（root 権限）が要る。
これは新規構築にあたるため SEAL §6 により私の権限外であり、採用するなら**先に裁定が要る**。

### (b) `twoder/operator.py` が stdlib を shadow する

py3.13 では `twoder/` を sys.path に載せた時点で `import operator` が
`twoder/operator.py` を拾い、`ImportError: cannot import name 'eq'` になる。
今回の計測はすべて分離ディレクトリで行った。本変更とは無関係の**既存問題**だが、
**FIX-01-OP で oracle 配管を触るときに必ず踏む**。配管更新の設計に織り込んでおくのが安全。
